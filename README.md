# Smythe University College AI

AI assistant web app supporting text, image, audio, and video uploads.

## Features
- Multi-model support: GPT-4, GPT-3.5, Gemini, Nano Banana
- Upload files: text, images, audio, video
- Chat history saved in localStorage
- Dark/Light mode toggle
- Download chat history

## Deployment
- **Netlify**: Place `chat.js` in `netlify/functions/`
- **Vercel**: Place `chat.js` in `api/`
- Set `OPENAI_API_KEY` as environment variable
